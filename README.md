# Cleaner
## 简介:

一个作为练手的使用JAVA8开发的图形化字典管理工具。使用SQLite数据库作为字典的存储。

## 功能

实现导出权重前N位字典内容，以及添加字典内容时如果重合，就在权重上加一。

Script文件夹中有简单的Python内容处理脚本。
